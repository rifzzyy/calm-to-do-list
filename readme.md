# 🌿 Green Calm To-Do List

A minimal, aesthetic, notebook-style to-do list built using **HTML, CSS, and JavaScript**.
Designed with a clean green theme to provide a calm and focused daily planning experience.

---

## ✨ Features

* 📓 **Notebook Style UI**

  * Tasks aligned like real paper lines
  * Minimal and aesthetic green design

* 📅 **7-Day Planner (Weekly System)**

  * Tasks are organized by day (Sunday → Saturday)
  * Each day has its own task list
  * Simple weekly structure for easy planning

* ⌨️ **Quick Add Tasks**

  * Press **Enter** to add tasks instantly
  * No extra buttons needed

* ✅ **Task Management**

  * Mark tasks as completed
  * Delete individual tasks
  * Clear all tasks for a selected day

* 💾 **Local Storage Support**

  * Tasks are saved in browser
  * Data persists even after refresh

* ✍️ **Daily Motivation Quote**

  * Changes based on selected day

---

## 🧠 How It Works

* The app stores tasks using a **7-day structure**:

* Each day holds its own tasks

* When a new week starts, the same structure is reused

---

## 🔁 Weekly Reset Behavior

* The app does **not automatically reset tasks**
* Since it uses a fixed 7-day system:

  * Tasks from previous week may still appear
* To reset:

  * Use the **clear (🗑) button** manually
